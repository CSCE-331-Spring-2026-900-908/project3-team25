A real restaurant-style starting point, not just a class demo screen.

This version is redesigned to feel closer to an actual modern kiosk and counter POS. The customer view emphasizes menu browsing and checkout flow, the cashier view uses a staff-side order ticket layout, and the menu board now looks like a lobby display instead of a plain page of cards.

Required interface launch

Customer kiosk Cashier screen Menu board Manager dashboard

Customer Kiosk

Designed like a self-order station with category chips, large menu cards, customization controls, and a sticky order summary.

  • Big touch targets
  • Drink customization area
  • Clear total and checkout

Cashier POS

Built more like a real counter terminal with category filtering, fast add buttons, live ticket summary, and customer-support widgets.

  • Separate staff layout
  • Order ticket view
  • Translation and assistant tools

Menu Board

Now looks more like an overhead digital board with large pricing, featured products, combo messages, and grouped menu sections.

  • High contrast display
  • Featured drinks panel
  • Non-interactive lobby view

Manager View

Kept as the internal desktop view for metrics, inventory alerts, and category breakdowns, separated from customer-facing ordering screens.

  • Back-office dashboard
  • Low-stock watchlist
  • Planning reminders

What changed from the first version

  • More restaurant-like color system and layout hierarchy
  • Customer ordering flow now has a real order summary area
  • Cashier view is now visually different from the kiosk
  • Menu board looks like a large display, not a basic page

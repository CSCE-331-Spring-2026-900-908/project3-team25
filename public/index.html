API Starter Features

Authentication

Mock login flow is included now so the site can run immediately. A clear hook is provided in the backend for replacing it with Google OAuth before final submission.

Machine Translation

Short menu text can be translated through the translation widget on the cashier and kiosk pages.

Weather Forecast

Weather data is used for seasonal drink suggestions. This can easily become a dynamic homepage recommendation feature.

Personal Assistant

A built-in assistant works right away with local fallback logic. If you add an OpenAI API key later, the assistant route can use a real LLM.

Important: the starter build is real and runnable, but OAuth credentials and final deployment wiring still need to be added by your team.
